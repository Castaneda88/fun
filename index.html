What are the 3 parts of an HTML element?

  1. opening tag <x>
  2. content(like this)
  3. end tag e.g. </x>

What Is A Void Element?

A void element is also known as a self closing tag i.g.

  • </img>

    • In this case the "/" may be omitted

  • <img>

    • This would effectively bare the same results

How To Create Paragraphs

In order to have paragraphs be formatted propperly, use the <p> and </p> closing tags. This will allow there to be linebreaks between one paragraph of content and the next.

We can see that any text inside the <p> and </p> tags are also wrapping, and take the shape of the area assigned. If no area is assigned, the paragraph takes the width of the browser window.

How To Create Headings

There are headers that range between <h1> to <h6>.

How To Create Bold Text

Bold text is accomlished using the <strong>text content</strong>

The <strong> element also effects screen readers to denote the specified text is important, indicating to screen readers of the texts importance, changing the readers tone of voice.

<strong> doesnt require any other elements to take its effect on a desired text, however, it will typically be used alongside other elements.

How To Create Italicized Text

to italicize text, we emphasize it using <em>emphasized content</em>. The <em> element also effects text that is read by a screenreader, giving "emphasis" to given text.

The Relationships Between Nested Elements

Parents, children, and grandchilren

Nested elements are elements that have been indented within other elements

Just like a directory
  1. Parent Direcory
    1. child
    2. child
      • grandchild
      • grandchild
    3. child

In the example shown above:

Notice the grandchild elements are nested inside the child line items, when put next to each other at the same level of indentation, they are considered siblings because their place of birth stems from the same origin. in this case child 2.

Just like real relationships, we can make the assertion that child 1, 2, and 3 are also siblings because their place of birth shares the "parent directory" as their origin.

How to Create HTML Comments

HTML comments are used to leave notes for other developers like a set of instructions. They are invisible to the web browser, but able to be seen in an editor.

In order to accomplish an html comment we use the triagle brackets to denote the opening and closing of the comment in the exaple below:

<!--This is an HTML comment-->

If you open the browsers inspector you may view the HTML comment in the code
