What are the 3 parts of an HTML element?

  2. content(like this)
  3. end tag e.g. </x>

What Is A Void Element?

A void element is also known as a self closing tag i.g.

  • </img>

    • In this case the "/" may be omitted

  • <img>

    • This would effectively bare the same results

How To Create Paragraphs

In order to have paragraphs be formatted propperly, use the <p> and </p> closing tags. This will allow there to be linebreaks between one paragraph of content and the next.

We can see that any text inside the <p> and </p> tags are also wrapping, and take the shape of the area assigned. If no area is assigned, the paragraph takes the width of the browser window.
