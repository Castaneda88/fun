How To Create Paragraphs

In order to have paragraphs be formatted propperly, use the <p> and </p> closing tags. This will allow there to be linebreaks between one paragraph of content and the next.

We can see that any text inside the <p> and </p> tags are also wrapping, and take the shape of the area assigned. If no area is assigned, the paragraph takes the width of the browser window.

How To Create Bold Text

Bold text is accomlished using the <strong>text content</strong>

The <strong> element also effects screen readers to denote the specified text is important, indicating to screen readers of the texts importance, changing the readers tone of voice.

<strong> doesnt require any other elements to take its effect on a desired text, however, it will typically be used alongside other elements.
